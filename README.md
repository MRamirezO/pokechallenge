## Instructions

- Clone this repo
- Open a terminal and cd into the project directory
- Run `npm install`
- Once finished downloading dependencies, run `npm start`
---------------------------------------------------------------------------------------------------------------

This exercise was made using create-react-app and some Material UI Core components
